delete_smartabase {neon}R Documentation

delete_smartabase

Description

Deletes events using event ID

Usage

delete_smartabase(
  url = NULL,
  username = NULL,
  password = NULL,
  event_id,
  cloud_mode = FALSE
)

Arguments

url

string: Smartabase url e.g. 'example.smartabase.com/site'

username

string: Smartabase username – ignore if setup with save_credentials()

password

string: Smartabase password – ignore if setup with save_credentials()

event_id

numeric: event IDs of events to be deleted

cloud_mode

boolean: if TRUE, no progress bar appears (i.e. for use in non-local, cloud environments)

Details

This function accepts Smartabase event IDs and deletes any records corresponding those event IDs vignette("deleting-data", package = "neon")

Value

Smartabase data

Examples

## Not run: 
# Delete one record with event_id = 999 from example.smartabase.com/site:
delete_smartabase(
  url = "example.smartabase.com/site",
  username = "john.smith",
  password = "examplePassword",
  event_id = 999
)

## End(Not run)
